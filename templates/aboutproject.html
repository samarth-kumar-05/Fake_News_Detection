About The Project

The idea of the project is to implement a machine learning algorithm to predict whether a given news is real or fake.
The dataset used here is taken from Kaggle and I have fitted a logistic regression machine learning model to the dataset
For this ML model the accuracy score for training data is around 98.65% and the accuracy score for test data comes out be be around 97.90% . I have used HTML , CSS , Tailwindcss , Bootstrap and JavaScript to build and style the web pages and then used Flask to convert this project to a web application.
